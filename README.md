#  Vite & Gourmand – Application de gestion traiteur

##  Présentation

Vite & Gourmand est une application web développée en PHP (architecture MVC) permettant la gestion complète d’une entreprise traiteur.

Elle permet :

- La consultation des menus
- La gestion des commandes clients
- La gestion des plats et allergènes
- La validation des avis clients
- La gestion des utilisateurs (Admin / Employé / Client)
- L’accès à des statistiques administrateur

---

##  Stack technique

- Front-end : HTML5, CSS3
- Back-end : PHP (PDO)
- Base de données : MySQL
- Serveur local : XAMPP
- Gestion de version : Git / GitHub
